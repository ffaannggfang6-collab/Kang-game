# Thai Kang Online Game
เกมไพ่แคงไทยแลนด์ ระบบ Real-time Multiplayer

## กติกาพิเศษ
- A เล็กสุด (1 แต้ม), J Q K = 10 แต้ม
- แต้มเท่ากัน: คนมี A ดอกจิก ชนะทันที
- ระบบไหล: J Q K จ่าย 1-2 เท่า

## การติดตั้ง
1. `npm install` ในทั้งฝั่ง client และ server
2. สร้างไฟล์ `.env` ตามคู่มือ
3. `npm start` เพื่อเริ่มเล่น
